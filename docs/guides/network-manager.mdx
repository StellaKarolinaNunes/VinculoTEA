Este guia é destinado a administradores master que gerenciam redes de ensino ou múltiplas sedes escolares.

## Configuração Inicial de Unidades

Para configurar suas primeiras unidades escolares, siga o fluxo administrativo:

1. Acesse o menu Unidades na barra lateral.
2. Clique em Adicionar Nova Unidade.
3. Preencha os dados institucionais (Nome Fantasia, CNPJ, Endereço).
4. Defina o Status como Ativo.

Repita este processo para cada uma das sedes físicas da sua rede.

## Importação e Gestão de Professores

A gestão eficiente do corpo docente é crucial para o funcionamento do VínculoTEA.

### Cadastro Individual
Para cadastros pontuais, utilize o botão Novo Professor no menu Professores, informando:
- Nome completo.
- E-mail institucional (usado para o login).
- Especialidade (ex: Psicopedagogia, AEE, Regente).

### Atribuição de Sedes
Um professor pode atuar em mais de uma unidade. No perfil do professor, utilize a seção Vínculos para associá-lo a múltiplas escolas. Isso permite que ele transite entre sedes usando uma única conta.

## Monitoramento da Rede

O Dashboard Master oferece uma visão consolidada:
- Total de alunos ativos na rede.
- Índice de PEIs atualizados por unidade.
- Alunos que aguardam avaliação inicial.

<Info>
  Utilize os filtros de unidade para gerar relatórios específicos de cada sede para reuniões pedagógicas municipais ou estaduais.
</Info>
