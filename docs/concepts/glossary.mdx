Este glossário visa alinhar a comunicação entre a equipe de tecnologia e os profissionais da educação.

## Termos Pedagógicos e de Saúde

### AEE (Atendimento Educacional Especializado)
Serviço da educação especial que identifica, elabora e organiza recursos pedagógicos e de acessibilidade para eliminar barreiras à inclusão.

### PEI (Plano Educacional Individualizado)
Documento que planeja as intervenções pedagógicas de forma personalizada para o aluno com deficiência, baseado em suas potencialidades e necessidades.

### CID (Classificação Internacional de Doenças)
Código universal que identifica doenças e problemas relacionados à saúde. No sistema, é usado para classificar formalmente o diagnóstico do aluno (ex: F84.0 para Autismo Infantil).

### Baseline (Linha de Base)
O nível atual de desempenho do aluno antes do início de uma intervenção. É o ponto de partida registrado nas primeiras etapas do PEI.

### PECS (Picture Exchange Communication System)
Sistema de comunicação por troca de figuras, muito comum para alunos com TEA não-verbais. O VínculoTEA permite registrar se o aluno utiliza este recurso.

## Termos Técnicos (TI)

### RLS (Row Level Security)
Segurança em Nível de Linha. Tecnologia de banco de dados que garante que um professor só consiga "ler" os dados dos alunos que pertencem à sua escola.

### JWT (JSON Web Token)
Um padrão de mercado para passar informações de autenticação com segurança entre o navegador e o servidor.

### CRUD (Create, Read, Update, Delete)
Acrônimo para as quatro operações básicas de um software: Criar, Ler, Atualizar e Deletar informações.

### Middleware
Código que executa entre a requisição do usuário e o processamento final, usado no sistema para checar se o usuário ainda está logado.
