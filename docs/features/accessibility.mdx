## Design para Neurodiversidade
Não acreditamos em "tamanho único". O VínculoTEA é a primeira plataforma de gestão escolar com um sistema de **Acessibilidade Dinâmica** nativo.

## Assistente de Configuração (Wizard)
Ao iniciar na plataforma, o usuário passa por um guia de 1 minuto para personalizar sua interface:

### Perfil Visual
- **Modo Escuro**: Redução de brilho para sensibilidade à luz.
- **Alto Contraste**: Legibilidade máxima.
- **Daltonismo**: Ajuste cromático para diferentes tipos de discromatopsia.

### Perfil Cognitivo / TDAH
- **Modo Foco**: Esconde elementos secundários da tela para reduzir distrações.
- **Tipografia**: Uso de fontes como *OpenDyslexic* para facilitar a leitura.

### Perfil Motor
- **Navegação por Teclado**: Shortcuts globais para usuários que não utilizam o mouse.
- **Áreas de Clique Ampliadas**: Facilitação para coordenação motora fina.

## Tecnologia Assistiva
- **Text-to-Speech (Sintetizador)**: O sistema pode ler conteúdos em voz alta para apoiar usuários com desafios visuais ou de leitura.
- **Comandos de Voz**: Execute ações como "Buscar Aluno" ou "Abrir Ajustes" apenas falando com o navegador.

## Como implementamos
Utilizamos uma arquitetura de **CSS Tokens** injetados via React Context. Ao contrário de plugins externos que "quebram" o layout, nossa acessibilidade é testada pixel a pixel para manter a harmonia visual.
